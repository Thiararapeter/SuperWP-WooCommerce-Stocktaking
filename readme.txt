A comprehensive stocktaking system for WooCommerce with count transfer, reset functionality, and detailed reporting.

== Description ==

superWP Woo Stocktake is a powerful inventory management solution for WooCommerce stores. It provides a robust set of tools for conducting accurate stocktakes, managing discrepancies, and generating detailed reports.

= Key Features =

* Conduct stocktakes with ease
* Transfer counted stock to WooCommerce
* Reset functionality for new stocktakes
* Detailed discrepancy analysis
* Audit and follow-up system
* Comprehensive reporting

= How It Works =

1. Initiate a stocktake
2. Count your inventory
3. Review discrepancies
4. Conduct audits and follow-ups
5. Generate detailed reports
6. Transfer final counts to WooCommerce

= Perfect For =

* E-commerce stores of all sizes
* Retailers with physical and online presence
* Businesses requiring regular inventory checks
* Companies needing detailed stock reports

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/superwp-woo-stocktake` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the WooCommerce -> Stocktaking menu to access the plugin's features.

== Frequently Asked Questions ==

= Is this compatible with the latest version of WooCommerce? =

Yes, superWP Woo Stocktake is regularly updated to ensure compatibility with the latest version of WooCommerce.

= Can I conduct partial stocktakes? =

Currently, the plugin is designed for full inventory stocktakes. Partial stocktake functionality may be added in future updates.

= How are discrepancies handled? =

Discrepancies are clearly highlighted in the stocktake report. You can review each discrepancy, provide reasons, and take follow-up actions through the audit system.

== Features in Detail ==

1. **Stocktake Initiation**: Easily start a new stocktake process with a single click.

2. **Counting Interface**: User-friendly interface for entering counted stock quantities.

3. **Discrepancy Analysis**: 
   - Automatically calculates discrepancies between expected and counted stock.
   - Provides visual indicators for positive and negative variances.

4. **Audit System**:
   - Allows for follow-up actions on discrepancies.
   - Supports custom actions and reason updates.

5. **Reporting**:
   - Generates comprehensive reports including:
     * Total expected vs counted stock
     * Total value of inventory
     * Number of SKUs with discrepancies
     * Detailed breakdown of discrepancies by product

6. **Stock Transfer**: 
   - Option to transfer counted stock to WooCommerce, updating product inventory levels.

7. **Reset Functionality**: 
   - Ability to reset counted stock, allowing for new stocktakes without residual data.

8. **Data Visualization**:
   - Presents stocktake data in an easy-to-understand format with cards and tables.

9. **User Roles and Permissions**:
   - Integrates with WordPress user roles for secure access to stocktake features.

== Technical Details ==

* Built for WordPress and WooCommerce
* Utilizes WordPress post types for storing stocktake data
* Leverages WooCommerce API for product data and stock updates
* Follows WordPress coding standards and best practices
